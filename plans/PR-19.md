## New model: `UnlockRule` (`unlock_rules`)
Fields:
- `id String @id @default(cuid())`
- `scopeType UnlockScopeType @map("scope_type")`
- `scopeId String @map("scope_id")`
- `ruleType UnlockRuleType @map("rule_type")`
- `ruleConfigJson Json @map("rule_config_json")`
- `isActive Boolean @default(true) @map("is_active")`
- `priority Int @default(0)`
- `createdAt DateTime @default(now()) @map("created_at")`
- `updatedAt DateTime @updatedAt @map("updated_at")`

Indexes:
- `@@index([scopeType, scopeId, isActive])`
- `@@index([ruleType])`
- `@@index([priority])`
- `@@map("unlock_rules")`

## New model: `UserUnlock` (`user_unlocks`)
Fields:
- `id String @id @default(cuid())`
- `userId String @map("user_id")`
- `scopeType UnlockScopeType @map("scope_type")`
- `scopeId String @map("scope_id")`
- `reason String?`
- `unlockedAt DateTime @default(now()) @map("unlocked_at")`

Relations:
- `user User @relation(fields: [userId], references: [id], onDelete: Cascade)`

Constraints/indexes:
- `@@unique([userId, scopeType, scopeId])`
- `@@index([scopeType, scopeId])`
- `@@index([userId, unlockedAt])`
- `@@map("user_unlocks")`

  - `scope_id: <http-basics-module.id>`
  - `rule_type: prereq_sections`
  - `rule_config_json: { section_ids: ["request-response-cycle"] }`

Idempotent approach:
- `deleteMany` existing rules for the seeded module scope + recreate deterministic rule set, or
